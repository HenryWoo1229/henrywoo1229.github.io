netty 是一个 Java 中用来做网络 NIO 的开源框架。提到 NIO 不得不先提一下网络 IO 框架

在 Java 中进行 Socket 开发时，JDK 提供了一个 IO 框架，服务端编写 java.net.ServerSocket，客户端编写 java.net.Socket，如果不了解的话，记住一点：此框架在 Server 要并发处理 Client 连接请求时，需要开启一个新的线程，当连接数变大后 Server 占用的资源就会增大，而线程间切换又会导致性能低下，所以复杂的场景不推荐使用

后来 JDK 提供了一个“非阻塞式” IO 框架，用来解决 IO 框架中的性能问题，由于这部分 API 比较复杂，导致业务代码写起来繁琐，所以有了 netty，对 NIO 做了封装，让它的开发变得简单
