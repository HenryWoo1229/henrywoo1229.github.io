    • 2020-04-20 在 Github 上用 hexo 打造自己的博客系统

    网上可以自由创作的博客网站挺多的，像 CSDN，掘金，简书，cnBlog 等等，那为什么还要在 GitHub 上打造一个自己的博客系统呢

    原因有二，其首为了装 X

    不开玩笑，第一点，体现了自己热爱技术钻研，喜欢动手实践的人设

    第二点，DIY 的东西，显得高大上一些

    那么本次从零开始，手把手带大家使用博客框架 Hexo 搭建一套真正属于自己的博客

    博客系统简单配置

    进入博客目录，说一下里面的文件构成（此目录基于 hexo 4.2.0 版本）

    _config.yml 跟博客站点配置相关，我们可以动手修改其中的参数，它主要分为几部分，挑几个重点的来说

    完整的说明查看官网 https://hexo.io/docs/configuration

    • Site

      title: Hexo 博客的标题，位于上图中央
      keywords: 博客的关键词，可以用多个，中间用逗号隔开
      author: John Doe 博客作者名字
      language: en 语言，中文的话可以改成 zh-cn

    • Extensions

      theme: landscape 设置要使用的主题目录，位于 themes 目录下

    • deploy

      type: 配置博客部署到远程服务系统上的类型，比如 git
